# Masalalar
## find01_max

  Berilgan sonlar ro'yxati bo'yicha ro'yxatdagi eng katta sonni qaytaring.

**1-misol:**

```Python
Input: [1, 2, 3, 4, 5]
Output: 5

```
**2-misol:**

```Python
Input: list1 = [12, 2, 5, 2, 7, 9, 1]
Output: 12

```
**Cheklovlar:**

  - 2 <= length(list1) <= 10^5
